# 🎵 Music Player

A simple and clean music player built using HTML, CSS, and JavaScript.

 🚀 Features

* ▶️ Play / Pause songs
* ⏭️ Next / Previous controls
* 📊 Progress bar with seek functionality
* 🎶 Displays current song name
* 🔁 Auto play next song

## 📁 Project Structure

```
music-player/
│── index.html
│── style.css
│── script.js
│── songs/
```

## 🛠️ Tech Used

* HTML
* CSS
* JavaScript

## 🎧 How to Use

1. Clone the repository
2. Open `index.html` in your browser
3. Click play and enjoy music 🎶
